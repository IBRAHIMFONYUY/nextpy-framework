Quick Start

# Install NextPy
pip install nextpy

# Create a new project
nextpy create my-app

# Start development server
cd my-app
nextpy dev

# Build for production
nextpy build

Why NextPy?

Python-First Development

Write your entire application in Python with familiar tools like FastAPI, Pydantic, and Jinja2. No JavaScript configuration needed.

Next.js-Inspired Architecture

File-based routing, SSR, SSG, and API routes - all the patterns you love from Next.js, now in Python.

Type Safety with Pydantic

Full type validation for API routes and data fetching with Pydantic models and Python type hints.

HTMX Integration

Client-side interactivity without heavy JavaScript frameworks. SPA-like experience with minimal overhead.
